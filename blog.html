Important FAQ

What is Document Object Model (DOM)?

The DOM (Document Object Model) in JavaScript is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to a part of the document (e.g., elements, attributes, text). JavaScript can interact with and manipulate this structure to dynamically change the content, style, and behavior of a webpage.

How do you select an Element from the DOM?

In JavaScript, there are several ways to select an element from the DOM. The method you choose depends on how the element is identified (e.g., by ID, class, tag, or more specific selectors).

What is event delegation in the context of DOM, and why it is useful?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element rather than attaching individual event listeners to each child element. This is possible because of event bubbling, where events triggered on a child element "bubble up" through its ancestors, allowing the parent to handle events from all of its child elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles in the DOM using JavaScript, you can use methods like getAttribute and setAttribute to retrieve and change attributes, and the style property to modify inline styles. For example, element.setAttribute('data-custom', 'newValue') changes an attribute, while element.style.color = 'red' updates the text color. To read computed styles, use window.getComputedStyle(element) which provides access to styles applied through CSS. This allows for dynamic updates to both attributes and visual presentation of elements in response to user actions or other events.
